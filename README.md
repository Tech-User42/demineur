## Important !
- Pour pouvoir utiliser ce projet correctement voici les étapes à suivre :
1. `git clone https://github.com/Tech-User42/demineur.git`
2. `cd demineur`
- Utilisez la commande adaptée à votre S.E :

- Windows `Code.exe`

- Linux `sudo chmod+x ./Code && ./Code`
